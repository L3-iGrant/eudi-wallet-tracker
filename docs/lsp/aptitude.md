# APTITUDE

Phase 2 Large Scale Pilot. Active. Focused on travel and mobile vehicle registration certificates.

## Scope

- Travel and Digital Travel Credentials (succeeds EWC)
- Mobile Vehicle Registration Certificate (mVRC)
- Cross-border interoperability tests for tickets and check-in

## Status

Active. Runs until 2027.
