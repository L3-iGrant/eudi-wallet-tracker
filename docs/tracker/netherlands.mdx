# Netherlands

<StatusBadge status="Public Pilot" />

57 residents of the Municipality of Amsterdam and the Municipality of Nijmegen tested the first version of the NL Wallet. The Dutch Government has publicly stated the initial release will not meet the full requirement set on time.
